# Soulver 3 — Notepad Calculator With Variables and Units

## About Soulver 3

Soulver 3 provides a smart notepad that calculates as you type — write equations in plain English, use variables, convert units, and see running totals. Soulver 3 combines word-based math with traditional calculation, live variable substitution, automatic unit conversion (currency, distance, weight, volume, temperature), and referenceable line-by-line results. Soulver 3 enables users to build complex spreadsheets without formulas — just type "150 * 2.5 hours" or "budget * 1.1" and get answers. Soulver 3 works offline, stores calculations as text, and outputs to Markdown, PDF, or plain text. ## Deep Dive — Notepad Calculator Notes

Natural language math — "forty-two percent of 500" or "yesterday + 3 weeks" or "USD 50 in EUR"  
Variables — assign values to words (budget = 1000, tax = 0.08), reuse anywhere, auto-update all references  
Unit conversion — length, area, volume, mass, temperature, time, speed, fuel economy, data size, currency (live rates)  
Running totals — sum of all numeric lines, or sum of selected lines, with automatic recalculation  
Line references — use @1, @2 to refer to previous line results, combine with variables  
Formatting — bold, italic, comments, section headers, export to Markdown, PDF, HTML
